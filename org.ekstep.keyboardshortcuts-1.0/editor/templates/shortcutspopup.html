ShortCut
                        Details
                    

                    

                        ctrl + x
                        Cut selected object
                    

                    

                        ctrl + c
                        Copy selected object
                    

                    

                        ctrl + v
                        Paste selected object
                    

                    

                        ctrl + a
                        Select all object
                    

                    

                        Delete
                        Delete selected object
                    

                    

                        ctrl + m
                        Add a new stage
                    

                    

                        ctrl + d
                        Duplicate a stage
                    

                    

                        ctrl + s
                        Save content
                    

                    

                        Home (for mac users use command+up)
                        Go to First Slide
                    

                    

                        End (for mac users use command+down)
                        Go to last Slide
                    

                    

                        ctrl + b
                        Make text style as bold
                    

                    

                        ctrl + i
                        Make text style as italic
                    

                    

                        ctrl + shift + >
                        Increase text size of selected text object
                    

                    

                        ctrl + shift + <
                        Decrease text size of selected text object
                    

                    

                        ctrl + alt + shift + l
                        Align a text object to left
                    

                    

                        ctrl + alt + shift + r
                        Align a text object to right
                    

                    

                        ctrl + alt + shift + e
                        Align a text object to center
                    

                    

                        ctrl + down arrow
                        Send an object to back
                    

                    

                        ctrl + up arrow
                        Send an object to front
                    

                    

                        up arrow
                        Move selected object up by 3 pixel
                    

                    

                        down arrow
                        Move selected object down by 3 pixel
                    

                    

                        left arrow
                        Move selected object left by 3 pixel
                    

                    

                        right arrow
                        Move selected object right by 3 pixel
                    

                    

                        shift + up arrow
                        Move selected object up by 1 pixel
                    

                    

                        shift + down arrow
                        Move selected object down by 1 pixel
                    

                    

                        shift + left arrow
                        Move selected object left by 1 pixel
                    

                    

                        shift + right arrow
                        Move selected object right by 1 pixel
                    

                    

                        ctrl + /
                        Show keyboard shortcut popup
                    

                

                
For mac users use command key in place of ctrl key
